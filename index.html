How to use the input

Type a Primitive Elvish word using these conventions:

  • Long vowels: Use macrons (ā, ē, ī, ō, ū) or type a colon after the vowel (a: → ā)
  • Morpheme boundaries: Use + to mark morpheme boundaries (e.g., aran+θorn)
  • Digraph shortcuts: Type uppercase digraphs to auto-convert (e.g., TH → θ, DH → ð)

Uppercase shortcuts

These UPPERCASE digraphs in the input are converted to phonetic symbols:

  • TH → θ
  • DH → ð
  • BH → β
  • CH → x
  • KH → x
  • GH → ɣ
  • PH → ɸ
  • NG → ŋ
  • GW → ƣ
  • HW → ʍ
  • SS → ſ

Lowercase digraphs (th, dh, etc.) are kept as-is.

Syllabic consonants

Use uppercase letter + underscore for syllabic consonants:

  • M_ → ṃ (syllabic m)
  • N_ → ṇ (syllabic n)
  • S_ → ṣ (syllabic s)
  • NG_ → ŋ̣ (syllabic ng)

Why single characters?

This app uses single characters to represent digraphs and other complex phonemes for ease of processing. Many phonetic symbols with diacritics (like kʰ or r̥) occupy multiple Unicode code points, which makes string manipulation unreliable. The single-character substitutes below are not meant to be IPA — they are simply practical placeholders that allow the rules to work correctly.

The main input is automatically converted to these characters, and the final output is converted back. Rule displays use the single-character forms throughout.

Symbol reference

  • θ = th (voiceless)
  • ð = dh (voiced)
  • β = bh (fricative b)
  • x = ch (spirant)
  • ɣ = gh (voiced velar)
  • ɸ = ph (spirant)
  • ŋ = ng (velar nasal)
  • ƣ = gw (labiovelar)
  • ʍ = hw (voiceless w)
  • ſ = ss (long s)
  • ꝁ = kʰ (aspirated k)
  • ƥ = pʰ (aspirated p)
  • ŧ = tʰ (aspirated t)
  • ꜧ = xʲ (palatalized)
  • ƕ = xʷ (labialized)
  • ꝉ = l̥ (voiceless l)
  • ꞧ = r̥ (voiceless r)
  • ᵯ = m̥ (voiceless m)
  • ꞃ = n̥ (voiceless n)
  • ṃ = syllabic m
  • ṇ = syllabic n
  • ṣ = syllabic s
  • ŋ̣ = syllabic ng
